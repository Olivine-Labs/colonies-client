colonies-client
===============

web client for colonies.

Installation
------------

* Install node.js `0.10.x`
* Clone project
* Run `npm install` inside project directory
* Run `npm install -g grunt-cli`

Running
-------

In one window: `grunt watch`
In another: `npm start`
